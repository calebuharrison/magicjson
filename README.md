# MagicJSON

A better (and vastly messier) compiletime JSON parser generator for Crystal.

## Installation

Add this to your application's `shard.yml`:

```yaml
dependencies:
  magicjson:
    github: [your-github-name]/magicjson
```

## Usage

```crystal
require "magicjson"
```
